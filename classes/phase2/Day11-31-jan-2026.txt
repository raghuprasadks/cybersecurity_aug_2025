MQTT Hands on 

1.Download and install mosquitto mqtt broker
	https://mosquitto.org/download/
Windows
mosquitto-2.0.18a-install-windows-x64.exe (64-bit build, Windows Vista and up, built with Visual Studio Community 2019

2.Verify whether mosquitto is running using windows search – services – mosquito
3. Stop the mosquitto service
4. Run mosquito from command prompt
	Cmd to C:\Program Files\mosquito

6. Subscribe to a topic
Open another terminal and cmd to C:\Program Files\mosquito
Type mosquitto_sub -h localhost -p 1883 -t "topic”

7. Publish a topic
Open another terminal and cmd to C:\Program Files\mosquito
Type mosquitto_pub -h localhost -p 1883 -t "topic" -m "Hello World“
8. Check the subscriber whether you have received the message
